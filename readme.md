## English

### 📝 Project Overview

This is a modern, high-performance landing page designed for a custom furniture and carpentry business. The goal was to blend traditional craftsmanship with modern web technologies, focusing on a smooth user experience and interactive elements.

### 🚀 Key Features

- **Infinite Scrolling Gallery:** A seamless, category-filtered project showcase powered by GSAP.
- **3D Interactive Cards:** Service sections with a 3D flip animation to reveal details on click.
- **Dynamic Filtering:** Users can filter past work by category (Kitchen, Wardrobes, Tables) without page reloads.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **Modern Animations:** Smooth transitions and GSAP-powered motion effects.

### 🛠 Tech Stack

- **HTML5 & CSS3:** Semantic markup and custom Grid/Flexbox layouts.
- **JavaScript (ES6+):** Logic for DOM manipulation and filtering.
- **GSAP (GreenSock Animation Platform):** Used for the high-performance infinite slider.
- **Font Awesome:** For professional iconography.

## Türkçe

### 📝 Proje Hakkında

Bu proje, özel tasarım mobilya ve marangozluk işletmeleri için tasarlanmış modern ve yüksek performanslı bir açılış sayfasıdır (landing page). Geleneksel işçiliği, akıcı bir kullanıcı deneyimi ve etkileşimli öğelerle birleştirerek modern web teknolojileriyle sundum.

### 🚀 Öne Çıkan Özellikler

- **Sonsuz Döngü Galeri:** GSAP kullanılarak hazırlanan, kategorilere göre filtrelenebilen kesintisiz proje vitrini.
- **3D Etkileşimli Kartlar:** Tıklandığında detayları gösteren 3D döndürme (flip) efektli servis bölümleri.
- **Dinamik Filtreleme:** Kullanıcıların iş örneklerini (Mutfak, Gardırop, Masa) sayfa yenilenmeden filtrelemesine olanak tanıyan yapı.
- **Duyarlı (Responsive) Tasarım:** Mobil, tablet ve masaüstü cihazlar için tam uyumluluk.
- **Modern Animasyonlar:** Akıcı geçişler ve GSAP destekli hareket efektleri.

### 🛠 Kullanılan Teknolojiler

- **HTML5 & CSS3:** Semantik yapı ve özel Grid/Flexbox tasarımları.
- **JavaScript (ES6+):** DOM manipülasyonu ve filtreleme mantığı.
- **GSAP (GreenSock):** Yüksek performanslı sonsuz kaydırma animasyonu için tercih edildi.
- **Font Awesome:** Profesyonel ikon setleri için kullanıldı.
